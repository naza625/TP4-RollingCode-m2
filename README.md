# 📘 Trabajo Práctico N°3 - JavaScript

## 📖 Descripción

En este trabajo práctico se desarrollan distintos ejercicios utilizando **JavaScript**, con el propósito de aplicar los conceptos fundamentales del lenguaje. A lo largo de las actividades se implementan soluciones que hacen uso de estructuras de control, arreglos, funciones, generación de números aleatorios y manipulación de datos, fortaleciendo la lógica de programación y la resolución de problemas.

## 🛠️ Tecnologías utilizadas

- HTML5
- JavaScript
